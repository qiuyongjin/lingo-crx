# Lingo

[English](README.md)

点击网页中的任意单词，即可在单词下方悬浮显示中文释义。

## 功能

- **点击查词** — 按住 Alt/Option 键点击任意网页中的单词，下方弹出中文释义
- **Ctrl 查词** — 或将光标悬停在单词上，按下并释放 Ctrl 键即可查词
- **AI 驱动** — 释义由 DeepSeek 大模型生成，准确简洁，附带音标和上下文例句
- **句子分段** — 整句按从句级别拆分为多个片段，每段配有独立翻译
- **语音朗读** — 点击扬声器图标听发音；可开启自动朗读，每次查词自动发音
- **查词历史** — 每次查词自动保存，包含单词、上下文和时间戳；通过侧边栏查看
- **悬浮按钮** — 页面右侧边缘有一个可拖拽的蓝色按钮，点击打开历史侧边栏
- **快捷设置** — 在弹窗中点击齿轮图标，可快速切换 Alt 键要求和自动朗读

## 安装使用

### 1. 安装扩展

1. 下载 `dist/chrome-mv3/` 目录
2. 打开 Chrome，进入 `chrome://extensions`
3. 开启右上角「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择 `dist/chrome-mv3/` 目录

### 2. 配置 API Key

1. 前往 [DeepSeek 开放平台](https://platform.deepseek.com/api_keys) 创建 API Key
2. 右键点击扩展图标 → 选项（或点击扩展图标 → 设置）
3. 粘贴 API Key，点击保存

### 3. 开始使用

- **Option+点击** 任意网页中的单词，释义即刻弹出
- **Ctrl** — 将光标悬停在单词上，按下并释放 Ctrl 键即可查词
- **点击弹窗外** 或按 Escape 键关闭弹窗
- **悬浮按钮** — 点击页面右侧的蓝色按钮打开历史侧边栏；可上下拖拽调整位置
- **扬声器** — 在弹窗中点击扬声器图标听发音
- **齿轮** — 点击齿轮图标切换设置（Alt 键要求、自动朗读）
